# Game of Thrones: Kingsroad TH

เว็บไซต์แฟนคอมมูนิตี้ภาษาไทยสำหรับเกม **Game of Thrones: Kingsroad**
รวมข่าวสาร สารจากผู้พัฒนา Roadmap ฐานข้อมูล คู่มือ และเครื่องคำนวณ

🌐 https://got-kingsroad-th.jcgameservice.workers.dev

## โครงสร้างโปรเจกต์

```
├── index.html        หน้าหลัก (สไลเดอร์ข่าว, สถิติ, อาชีพ, ไทม์ไลน์)
├── news.html         ข่าวสารและสารจากผู้พัฒนา
├── roadmap.html      แผนการอัปเดตแต่ละซีซัน
├── database.html     ฐานข้อมูล อาชีพ / บอส / ไอเทม / แผนที่
├── guide.html        คู่มือผู้เล่นใหม่และผู้เล่นระดับสูง
├── calculator.html   เครื่องคำนวณ (ภาษี, เวลาฟาร์ม, กำไร)
├── css/style.css     สไตล์ทั้งหมด (ธีมดำ-ทอง GoT)
├── js/app.js         สคริปต์ (สไลเดอร์, ค้นหา, เมนูมือถือ ฯลฯ)
└── images/           โลโก้ ภาพข่าว และพื้นหลัง
```


## ✏️ วิธีแก้ไขเว็บ (ฉบับเร็ว)

ตอนนี้ **เมนูและ footer อยู่ที่ไฟล์เดียว** ไม่ต้องไล่แก้ทุกหน้าแล้ว:

| อยากแก้อะไร | แก้ที่ไฟล์ไหน |
|---|---|
| เมนูบน / เพิ่ม-ลบหน้า / footer | `js/layout.js` (แก้ตัวแปร `MENU` และ `FOOTER_HTML` ด้านบนสุด) |
| สี ฟอนต์ ดีไซน์ทั้งเว็บ | `css/style.css` |
| สไลเดอร์ ค้นหา ปุ่มต่างๆ | `js/app.js` |
| เนื้อหาแต่ละหน้า | ไฟล์ `.html` ของหน้านั้น (เหลือแต่เนื้อหา สั้นลงมาก) |
| ข้อมูลอาชีพ | `classes.html` |

ตัวอย่าง: อยากเพิ่มเมนูใหม่ → เปิด `js/layout.js` → เพิ่มบรรทัดใน `MENU` → เซฟ → ขึ้นครบทุกหน้าทันที
